Tentang Saya

Perkenalkan, nama saya Rifa Eka Prasetya, seorang mahasiswa Teknik Informatika di Universitas Nahdlatul Ulama Sunan Giri. Sejak kecil, saya sudah memiliki ketertarikan besar terhadap dunia koding yang terus saya tekuni hingga kini. Saat ini, saya telah mempelajari dasar-dasar pemrograman web seperti HTML dan CSS sebagai langkah awal dalam perjalanan saya di bidang teknologi. Saya tinggal di Bojonegoro, Jawa Timur, Indonesia. Saya memiliki cita-cita untuk menjadi seorang freelancer programmer sekaligus guru kejuruan, agar bisa berbagi ilmu dan menginspirasi generasi muda di bidang teknologi.
